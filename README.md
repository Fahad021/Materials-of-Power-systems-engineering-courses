Catalogue of openly available (no-login) graduate power-systems courses from top North American universities, where course materials are provided as downloadable files (lecture notes, assignments, and/or exams).

## Inclusion rules used

- North American top-ranked university in electrical/power engineering.
- Graduate-level (masters/phd) power or closely related power-engineering course.
- No authentication/paywall required to access course materials.
- Course pages include downloadable instructional materials.
